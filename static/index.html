About This Demo

This demo showcases a complete Model Context Protocol (MCP) implementation with multiple layers:

Architecture

  1. Frontend: This web interface
  2. Chat Backend: LLM with tool calling (OpenAI)
  3. MCP Server: Exposes user search as a tool
  4. FastAPI Backend: REST API with mock data

How It Works

When you ask a question in the chat, the system:

  1. Sends your message to the LLM (OpenAI)
  2. LLM decides if it needs to call the search tool
  3. Tool makes API request to fetch user data
  4. Results are sent back to the LLM
  5. LLM formats a natural language response
